Hubungi Admin
1

Catat Hasil Panen

Buka halaman "Catat" dari menu navigasi, lalu masukkan jumlah tandan dan pilih lokasi blok panen. Tekan tombol "Simpan Data Panen" untuk menyimpan data.

2

Lihat Statistik Harian

Buka halaman "Beranda" untuk melihat total panen hari ini dan statistik mingguan secara ringkas. Data akan otomatis diperbarui.

3

Periksa Riwayat Panen

Buka halaman "Riwayat" untuk melihat catatan panen sebelumnya. Data dikelompokkan berdasarkan hari untuk memudahkan pencarian.

4

Hubungi Admin

Jika Anda mengalami kendala atau memiliki pertanyaan seputar penggunaan aplikasi, silakan hubungi admin melalui tombol di bawah.

FAQ - Pertanyaan Umum

Apakah data saya disimpan secara aman?

Ya, semua data panen Anda disimpan dengan aman di server kami dan hanya dapat diakses oleh Anda dan administrator yang berwenang.

Bagaimana jika saya salah input data?

Anda dapat menghubungi administrator untuk melakukan koreksi data yang telah disimpan.

Apakah aplikasi ini bekerja tanpa koneksi internet?

Saat ini aplikasi memerlukan koneksi internet untuk menyimpan dan mengakses data. Kami sedang mengembangkan fitur offline untuk versi mendatang.
